Project

EnviroHub is a cheap environmental monitoring toolkit that allows users to accurately gather and view a variety of environmental metrics. Similar systems can cost thousands of dollars, which was the motivation for this product [1].The product was designed with a public API in order to enable users to create and integrate their own environmental data devices. All of the code is open source to encourage expansion and improvement of the product. EnviroHub makes it simple and straightforward for users to see registered device locations on our website. Users can click on their device on a map and see all recent data associated with that device. Data from any specific device is charted so that users can see change over time and identify any issues with local environmental quality metrics. Data charts can be filtered by start and end dates and display a log of the latest recordings from the device. The currently developed hardware was designed and assembled with affordable and publicly accessible parts. The device created in this course was Arduino-centric.

An EnviroHub web server can be easily installed on a Debian/linux machines. We thought that it was important to allow users to install our system on their own machines for privacy and security reasons. We know that data security and integrity is important for our users.

Hardware

The device created in this course cost $244, and was capable of monitoring water pH and temperature, and submitting that data wirelessly to the server. Other Arduino compatible sensors can be purchased and integrated with the system that we have created. With modifications to the device website, Envirohub will then be capable of intaking and displaying data collected from any added sensors.
